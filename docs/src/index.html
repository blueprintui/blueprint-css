Blueprint Layout

A lightweight layout library for building great responsive mobile first UIs that work everywhere. Open Source, built with CSS Grid and Flexbox.

  • Easy to use declarative layouts
  • Grid, block, and inline layouts
  • Responsive via container queries
  • Only ~3kb of CSS

Flexible Grid

The 12 column grid enables flexible and responsive layouts. Built on the latest Container Queries spec, Blueprint Layouts enable your UI components to work in any context not tied to a specifc screen size.
