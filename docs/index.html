Blueprint Layout

A lightweight layout library for building great responsive mobile first UIs that work everywhere. Open Source, built with CSS Grid and Flexbox.

Examples Github

Getting Started

Blueprint Layout provides easy to use layout utilities for the Web.

  • responsive
  • grid, block, and inline layouts
  • built on latest container quieries
<div bp-layout="grid gap:md cols:6">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>
