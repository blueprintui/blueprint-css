Gaps allow control of white space between layout items. Gaps can be set via t-shirt sizes, `xs`, `sm`, `md`, `lg`, `xl`.

## Extra Small

<div bp-layout="grid gap:xs cols:3" demo>
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>

```html
<div bp-layout="grid gap:xs cols:3">
  <div>1</div>
  <div>2</div>
  <div>3</div>
  <div>4</div>
</div>
```
